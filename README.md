# AI-Image-Generation-Using-Stable-Diffusion
A Python project that generates high-quality images from natural language prompts using Stable Diffusion and the Hugging Face Diffusers library. Supports single and batch image generation with GPU acceleration.

# 🎨 AI Image Generation using Stable Diffusion

An AI-powered image generation project built with **Python** and the **Stable Diffusion** model from the Hugging Face Diffusers library. This project converts natural language prompts into high-quality images using state-of-the-art text-to-image diffusion models.

Designed for learning and experimentation, the notebook demonstrates how to install the required libraries, load a pre-trained Stable Diffusion pipeline, generate images from custom prompts, and save the generated outputs automatically.

## ✨ Features

* 🖼️ Generate realistic AI images from text prompts
* 🤖 Powered by Stable Diffusion and Hugging Face Diffusers
* ⚡ GPU acceleration with CUDA support (when available)
* 📝 Supports both single and multiple prompt generation
* 💾 Automatically saves generated images
* 🔧 Simple, beginner-friendly Python implementation
* 🚀 Easy to customize with your own prompts

## 🛠️ Technologies Used

* Python
* Stable Diffusion
* Hugging Face Diffusers

* Transformers
* Accelerate

## 📌 Project Objective

The goal of this project is to demonstrate how modern generative AI models can transform natural language descriptions into high-quality images. It serves as a practical introduction to text-to-image generation using Stable Diffusion and provides a foundation for building AI art generators, creative design tools, and generative AI applications.
